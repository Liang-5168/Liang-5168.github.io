AS#8 — Video Storytelling Project  
Copilot Prompt Summary

這份紀錄說明我如何使用 Copilot 協助完成一個以捲動驅動（scroll‑driven）的影片敘事網站。主要需求包括：全螢幕背景影片、分章節的 sticky 影片呈現、左側 2×2 縮圖預覽與右側長文導覽，以及捲動觸發播放和頂部進度條。Copilot 協助我產出 HTML/CSS 範本與主要 JavaScript 行為建議，網站目的是讓使用者透過慢速捲動「進入」四個影像切面，體驗紅包場的場域與情緒。

1. Landing Page Prompts

- 版面：高度 100vh，置中對齊（flex column），文字置中。
- 背景：深色 radial‑gradient（上方光源至暗色）。
- 文字：系統字體、白色、letter‑spacing 明顯，標題使用 clamp() 支援 RWD。
- 內容：landing‑tag、主標題「留聲的房間」、描述段落（介紹作品概念與四段影片）。
- CTA：圓角膠囊按鈕（id="goOnboarding"），click → scrollIntoView({ behavior: "smooth" })。
- Scroll hint：底部 SCROLL 提示與垂直動畫（keyframes）。
- 其他：禁止水平滾動（overflow‑x:hidden）、深色對比與間距調整。

2. Onboarding Page Prompts

- 左側：2×2 影片縮圖格（.video-grid），整組為 sticky（position:sticky; top:0; height:100vh）。
- 縮圖結構：.thumb-card 包含 <video muted preload="metadata"> 與遮罩文字（label/title/note）。
- 互動：
  - hover：video.currentTime = 0; video.play(); mouseleave → pause + currentTime = 0。
  - click：data-target → 對應章節平滑捲動。
- 右側：onboard‑text 長文說明，與左側 sticky 預覽形成閱讀節奏。
- 影片屬性與樣式：muted、preload="metadata"、灰階過渡、hover 放大。
- RWD：窄螢幕改為單欄，縮圖變一列並調整高度。

4. Technical Requirements

- 必要 <video> 屬性：
  - muted、preload="metadata"（縮圖與章節影片）；章節影片支援 loop。
  - 自動播放需 muted（部分瀏覽器限制非靜音自動播放）。
- Sticky layout：左側預覽與章節影片使用 position:sticky; top:0; height:100vh。
- Full‑screen video：width:100%; height:100%; object-fit:cover。
- Smooth scroll：使用 Element.scrollIntoView({ behavior: "smooth" })。
- CSS 結構（建議類別）：
  - .landing / .onboarding / .chapters-wrapper / .chapter-outer / .chapter-video / .chapter-text / .video-grid / .thumb-card / .video-controls / .progress-strip / .progress-bar-inner
  - 使用 clamp()、media queries（max-width:992px, 576px）。
- HTML 組成（必要 DOM）：
  - 進度條 .progress-strip > #scrollProgress
  - 影片控制按鈕 #playAll #pauseAll #toggleMute
  - Landing、Onboarding、四個章節（id=chapter1..4）、footer
  - 縮圖 .thumb-card data-target 對應章節 id
- JavaScript 要點：
  - 縮圖 hover 播放(mouseenter/mouseleave)、click 平滑捲動。
  - 全域播放/暫停/靜音（遍歷章節影片）。
  - 捲動時更新進度條（window.scrollY / docHeight）。
  - 依章節位置 (getBoundingClientRect) 判定播放/暫停。
  - 監聽 scroll 與 resize，並在載入時執行 handleScroll()。
